## Dataset description

This dataset contains images frequently used in computer vision research. 

## Data source
The images have been downloaded from the middlebury dataset at the url: https://vision.middlebury.edu/flow/data/

## Files list
The raw image files are:
- dimetrodon.png
- hydrangea.png
- lena_color.png
- mandrill.tiff
- peppers3.tif
- rubber_whale.png
